**作成日：2026年4月18日**
**次のステップ：Claude Codeで実装開始**

## 1. プロジェクト概要

### サイト名
**amico stitch library**

### キャッチコピー
> 世界の手芸をつなぐ

### サブタイトル
> the knitting stitch library

### コアコンセプト
編み物の「なぜ？」に、静かに答える場所。
記号の意味（What）→ 編み地への作用（Why①）→ 使いどころ（Why②）という流れで理解できる、日英韓三言語対応の編み図記号辞書。

### ターゲット
- 編み物を始めたばかりの人（日本語ユーザー）
- 日本の編み図を読みたい海外のニッター（英語・韓国語ユーザー）

---

## 2. サイト構成

```
amico
│
├── 段数カウンター（独立アプリ・別ページ）
│     毎回使うツール。シンプル・すぐ開ける設計。
│     ※すでに別途ローンチ済み
│
└── amico stitch library（メインサイト）
      ├── 編み図記号辞書（Index）
      │     記号を一つ一つ引く。カード形式。
      │     「この記号はなに？」に答える。
      │
      ├── 学習資料（Guide）※将来追加
      │     体系的に理解する。グループでまとめて解説。
      │     「増し目ってどんな種類があるの？」に答える。
      │
      ├── 針のサイズ変換ツール
      │     日本号数 ↔ mm ↔ 海外サイズ
      │     辞書の記号ページと連携予定
      │
      ├── 毛糸の太さガイド
      │     レース・合細・並太・極太
      │     「この編み方に向いている糸」と辞書をリンク
      │
      └── おすすめの本・道具（Amazonアフィリエイト）
            記号・糸・針に文脈で自然に紐づける
            ※コンテンツに沿った自然な埋め込みで信頼感を保つ
```

## 3. デザイン仕様

### カラーパレット（確定）
比率：ベース7割 ／ メイン2.5割 ／ アクセント0.5割

| 役割 | カラーコード | 用途 |
|------|-------------|------|
| 文字（ダーク） | `#343830` | 見出し・強調テキスト |
| 本文 | `#626858` | 本文テキスト全般 |
| 補足 | `#989E90` | サブテキスト・ラベル |
| 背景 | `#F0EFE8` | ページ背景・カード背景 |
| メイン（グリーン） | `#7DB8A8` | ボーダー・バッジ・アクセント背景 |
| メイン淡 | `#E4F2EE` | カード内背景・検索バー |
| メイン罫線 | `#B0D4CA` | 仕切り線・枠線 |
| アクセント（テラコッタ） | `#E8A898` | 関連ボタン・ロゴドット・ポイント |
| アクセント淡 | `#FAF0EE` | アクセントボタン背景 |

### デザイン方針
- 文字はすべてグレー系で読みやすく
- グリーンはバッジ・ボーダー・背景アクセントとして2.5割で使用
- テラコッタは「これも覚えたい」ボタン・ロゴドットのみ（0.5割）
- 華奢すぎない、骨格のしっかりしたデザイン
- 堅苦しくなく、ものづくりが楽しくなる雰囲気

### ロゴ
```
amico •
the knitting stitch library
```
（ドット部分はテラコッタ #E8A898）

### フォント方針
- システムフォント優先（日本語・英語・韓国語の多言語対応）
- 見出し：font-weight 500
- 本文：font-weight 400、line-height 1.75

## 5. 記号一覧 v1（三言語対照）

### 基本（Basic）

| 略称 | 日本語 | English | 한국어 |
|------|--------|---------|--------|
| K | 表目 | Knit | 겉뜨기 |
| P | 裏目 | Purl | 안뜨기 |
| CO | 作り目 | Cast On | 코잡기 |
| BO | 伏せ目 | Bind Off / Cast Off | 코막음 |

### 増し目（Increase）

| 略称 | 日本語 | 別称 | English | Alt | 한국어 | 特徴 |
|------|--------|------|---------|-----|--------|------|
| YO | 掛け目 | | Yarn Over | | 바늘비우기 / 걸기코 | 穴が開く増し目 |
| M1 | ねじり増し目 | 向き指定なし | Make 1 | | 코 늘리기 | シンカーループをねじる |
| M1R | ねじり増し目（ループ右上） | 右ねじり増し目 | Make 1 Right | | 오른코 늘리기 | 右に傾く・穴なし |
| M1L | ねじり増し目（ループ左上） | 左ねじり増し目 | Make 1 Left | | 왼코 늘리기 | 左に傾く・穴なし |
| RLI | 右増し目（引き上げ） | | Right Lifted Increase | KRL | 오른쪽 끌어올려 늘리기 | 前段を引き上げ・より目立ちにくい |
| LLI | 左増し目（引き上げ） | | Left Lifted Increase | KLL | 왼쪽 끌어올려 늘리기 | 前段を引き上げ・より目立ちにくい |
| KFB | 編み出し増し目 | | Knit Front and Back | kfb | 겉뜨기 코늘림 | 1目から2目編み出す |

### 減らし目（Decrease）

| 略称 | 日本語 | English | 한국어 | 特徴 |
|------|--------|---------|--------|------|
| K2tog | 右上2目一度 | Knit 2 Together | 오른코 2코 모아뜨기 | 右に傾く |
| SSK | 左上2目一度 | Slip Slip Knit | 왼코 2코 모아뜨기 | 左に傾く |
| S2KP | 中上3目一度 | Slip 2, K1, Pass 2 Slipped Over | 가운데 3코 모아뜨기 | 中央に向かって傾く |
| K3tog | 右上3目一度 | Knit 3 Together | 오른코 3코 모아뜨기 | 右上に3目を一度に |

### その他（Other）

| 略称 | 日本語 | English | 한국어 |
|------|--------|---------|--------|
| Sl | すべり目 | Slip | 미끄러뜨기 |
| tbl | ねじり目 | Through Back Loop | 뒤고리뜨기 / 꼬아뜨기 |
| PM | マーカーを入れる | Place Marker | 마커 넣기 |

### ⚠️ 重要な検証メモ

**「右増し目」という日本語表記は曖昧**
- 文脈によってM1RとRLIのどちらも指せてしまう
- 辞書では以下のように使い分ける：
  - M1R → 「ねじり増し目（ループ右上）」
  - RLI → 「右増し目（引き上げ）」

## 6. 検索機能仕様

- 日本語・英語・韓国語すべてで検索可能
- 記号の略称（M1R）でも検索可能
- カテゴリーフィルター：すべて・増し目・減らし目・透かし編み・基本
- 言語切り替え：日本語 / English / 한국어

## 7. イラスト方針

**現在**：SVGで概念図をプレースホルダーとして表示
**将来**：手描きイラストをスキャン・撮影して差し替え

各カードに2種類のイラストを配置：
1. **編み地の仕上がり**：メリヤス地の中で記号がどう見えるか
2. **使いどころのイメージ**：ラグランラインなど、実際の使用場面の構造図

動画：YouTubeを埋め込み（出典明記）。将来的に自前動画に移行。

## 8. 収益化方針

- Amazonアフィリエイト：コンテンツに沿った自然な埋め込み

## 9. 技術スタック（推奨）

### フロントエンド
- HTML / CSS / JavaScript（シンプルな静的サイト）
- または Next.js（将来の拡張を考えると推奨）

### データ管理
- 記号データはJSONファイルで管理
- `/data/symbols.json` に全記号データを格納
- 画像は `/images/symbols/` に格納

### ホスティング
- GitHub Pages（無料・シンプル）
- または Vercel（Next.js使用時）

### ファイル構成（案）
```
amico-stitch-library/
├── index.html          # トップ・検索ページ
├── symbol/
│   └── [id].html       # 各記号の詳細ページ
├── guide/              # 学習資料（将来）
├── data/
│   └── symbols.json    # 全記号データ
├── images/
│   └── symbols/        # 記号イラスト
├── css/
│   └── style.css       # スタイルシート
└── js/
    └── main.js         # 検索・フィルター機能
```
