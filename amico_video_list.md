# amico stitch library — 動画URLリスト

記号ごとにYouTube動画URLを記入して、Claude Codeに渡してください。

---

## 使い方

1. 各記号の「動画URL」欄にYouTubeのURLを記入
2. 確認欄に ✅ をつける
3. 以下のテキストをClaude Codeに貼り付ける：

```
以下のURLを各記号の動画に埋め込んでください。

M1R: https://...
M1L: https://...
（以下リスト）
```
